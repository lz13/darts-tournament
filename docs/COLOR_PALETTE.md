## Color Palette Summary

For future reference, here's the dark theme color palette we're using:

| Element        | Tailwind Class     | Hex               |
|----------------|--------------------|-------------------|
| Background     | `bg-gray-950`      | `#030712`         |
| Cards/Sections | `bg-gray-900`      | `#111827`         |
| Borders        | `border-gray-800`  | `#1f2937`         |
| Muted text     | `text-gray-400`    | `#9ca3af`         |
| Primary accent | `bg-emerald-600`   | `#059669`         |
| Primary hover  | `bg-emerald-500`   | `#10b981`         |
| Primary light  | `text-emerald-400` | `#34d399`         |
| Losers bracket | `bg-amber-*`       | amber tones       |
| Grand final    | `bg-yellow-*`      | yellow/gold tones |
